# image2tiled
Tool that creates a tiled map based on an image that describes a built map.

## Usage

`$ image2tiled --tile-size 16 map.png`

Alternatively
`$ python -m image2tiled --tile-size 16 map.png`

## Installation

`pip install image2tiled`
